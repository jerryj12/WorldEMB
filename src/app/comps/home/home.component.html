FILE FORMATS


JPG – we use this format because it is really a good choice for images with photos, numerous colors and scanned images.
PNG – this format is ideal for digitized designs with colors, mostly larger areas of solid colors.
PDF – This format is our favorite especially if it is a vector file. By using PDF, the artwork can easily be scaled without losing the actual quality.
EPS, AI, SVG – These are also well known vector formats that we prefer. But for these formats, you should convert fonts into outlines.
